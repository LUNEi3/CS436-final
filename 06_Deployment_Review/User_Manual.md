# คู่มือการใช้งาน (User Manual)
## Definition Dash


### คำแนะนำสำคัญ
*แอปพลิเคชันนี้ออกแบบมาสำหรับ **Mobile View** กรุณาใช้งานผ่านมือถือหรือเปิด Developer Tools (F12) บน Browser แล้วเลือกโหมด Mobile Device เพื่อการแสดงผลที่ถูกต้อง*

### 1. การเริ่มต้นใช้งาน
1.  เข้าสู่เว็บไซต์ Link: https://definition-dash-liart.vercel.app/
2.  จะพบหน้าจอหลัก (Welcome Screen)
3.  กดปุ่ม **"Get Start"** (หรือปุ่มเริ่มเกม) เพื่อเข้าสู่การทดสอบ
4.  จะพบหน้าจอสำหรับกรอกชื่อ (ห้ามใช้ภาษาอื่นนอกจาก ภาษาอังกฤษ)


### 2. การทำแบบทดสอบ
1.  อ่านคำที่ปรากฏ (Word) ที่ปรากฏบนหน้าจอ
2.  เลือกคำศัพท์ที่เป็น Synonym ของคำที่ปรากฏ (Choice) ที่ถูกต้องที่สุด
3.  สังเกตเวลาที่นับถอยหลังด้านบน
4.  หากตอบถูกเวลาจะเพิ่มขึ้น 2 วินาที และ <u>ได้คะแนน</u> 1 คะแนน
5.  หากตอบผิดเวลาจะลดลง 3 วินาที และ <u>ลบคะแนน</u> 1 คะแนน

### 3. การดูผลลัพธ์
1.  เมื่อหมดเวลาหรือทำครบทุกข้อ ระบบจะพาไปที่หน้าสรุปผล (Result Page)
2.  หน้าจอจะแสดงคะแนนที่คุณทำได้
3.  กดปุ่ม **"Back"** หากต้องการเริ่มเล่นใหม่อีกครั้ง
