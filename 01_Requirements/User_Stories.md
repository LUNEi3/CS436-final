# User Stories
## โครงการ: Definition Dash

| ID | As a (User Role) | I want to (Action) | So that (Benefit/Value) |
| :--- | :--- | :--- | :--- |
| **US-01** | ผู้ใช้งาน (Guest) | เริ่มทำแบบทดสอบได้ทันที | ฉันจะได้ไม่ต้องเสียเวลาในการสมัครสมาชิก |
| **US-02** | ผู้ใช้งาน (Guest) | เห็นตัวนับเวลาถอยหลังระหว่างทำข้อสอบ | ฉันจะได้บริหารเวลาในการตอบคำถามได้ทัน |
| **US-03** | ผู้ใช้งาน (Guest) | เห็นสรุปคะแนนเมื่อทำข้อสอบเสร็จ | ฉันจะได้รับรู้ความสามารถของตนเอง |
| **US-04** | ผู้ใช้งาน (Guest) | ใช้งานผ่านหน้าจอมือถือได้สะดวก | ฉันสามารถเล่นเกมได้ด้วยมือเดียวหรือขณะเดินทาง |
| **US-05** | ผู้ใช้งาน (Guest) | เริ่มเล่นเกมใหม่ (Restart) ได้เมื่อจบเกม | ฉันสามารถแก้ไขคะแนนหรือลองเล่นใหม่ได้ทันที |
